### 💻 Featured Projects

#### [🧠 Your-Own-AI (VectorDB)](https://github.com/Shashank17singh/Your-Own-AI)
**Engineered a high-performance vector database from scratch in C++**, optimizing search via HNSW and KD-Tree index structures against cosine, Euclidean, and Manhattan metrics. Built a full local RAG pipeline via Ollama by chunking documents into overlapping segments, generating 768D embeddings, and retrieving optimized context for local Llama 3.2 inference.

#### [📡 DPI Engine (Deep Packet Inspection)](https://github.com/Shashank17singh/DPI-Engine)
**Architected a high-throughput C++17 Deep Packet Inspection (DPI) engine** capable of reconstructing TCP/UDP flows and classifying traffic using TLS SNI and HTTP Host parsing. Implemented single-threaded and multi-threaded (load-balancer + fast-path) architectures to selectively block traffic by app, IP, or domain at scale.

#### [🏠 Home Price Suite](https://github.com/Shashank17singh/Home-Prices-API)
**Deployed an end-to-end ML pipeline for predicting home prices**, serving a Scikit-learn regression model (trained via K-Fold CV) through a production-ready REST API using Flask. Automated CI/CD pipelines via GitHub Actions to deploy the application on AWS EC2 using Docker and an Nginx reverse proxy. [**Live Demo**](https://home-prices-api.duckdns.org)

#### [📡 UPI Mesh (Offline Payments)](https://github.com/Shashank17singh/UPI_Mesh)
**Built a resilient payment backend for offline transactions** using hybrid RSA/AES-GCM encryption, a Bluetooth mesh simulator, and idempotent settlement via FastAPI, simulating secure financial routing without internet connectivity. [**Live Demo**](https://upi-mesh.duckdns.org/)
